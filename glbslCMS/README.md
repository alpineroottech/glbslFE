# Gurans CMS - Sanity Studio

Content management system for Gurans Laghubitta Bittiya Sanstha Limited.

## Setup

1. Copy `.env.example` to `.env` and fill in your Sanity project details
2. `npm install`
3. `npm run dev` — opens Studio at http://localhost:3333

## Deployment

```bash
npm run build    # Build static Studio
npm run deploy   # Deploy to sanity.studio
```
